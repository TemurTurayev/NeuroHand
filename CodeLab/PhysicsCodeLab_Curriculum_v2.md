### 1.1 Принципы построения курса

#### Конструктивизм Пиаже
Знание не передаётся — оно конструируется учеником через опыт. Каждая задача создаёт когнитивный конфликт (ожидание vs реальность симуляции), который ведёт к обучению.

#### Зона ближайшего развития (Выготский)
Каждая задача чуть сложнее предыдущей. Подсказки — это "леса" (scaffolding), которые убираются по мере роста компетенции.

#### Спиральный учебный план (Брунер)
Ключевые концепции возвращаются на каждом уровне с возрастающей глубиной:
- Уровень 1: Функция — это правило y = 2x

#### Принцип Alan Becker: "Математика живая"
Вдохновлённые серией "Animation vs Math":
- Функции и фигуры — это персонажи с характером
- Уравнения решаются через взаимодействие, не через зубрёжку
- Графики можно "трогать", деформировать, комбинировать
- Ошибка — это не провал, а неожиданное поведение персонажа

#### Активное обучение
Соотношение теория:практика = 20:80. Теория даётся только когда появляется потребность.

**Сюжет**: *Познакомься с семьёй линейных функций. У них общие черты, но каждый уникален.*

```

│                                                                 │
│         y                                                       │
│         │    ╱ k=2 (энергичный)                                │
│         │   ╱                                                   │
│         │  ╱  ╱ k=1 (спокойный)                                │
│         │ ╱  ╱                                                  │
│         │╱  ╱   ── k=0 (ленивый, горизонтальный)               │
│    ─────┼─────────                                              │
│        ╱│╲                                                      │
│       ╱ │ ╲ k=-1 (идёт в другую сторону)                       │
│         │                                                       │
│         │   b = сдвиг вверх/вниз (где пересекает ось Y)        │
│                                                                 │
│   🎮 СЛАЙДЕРЫ:                                                  │
│   k (наклон):    [-3 ═══●═══ +3]  текущее: 1.5                 │

│                                                                 │
```

**Сюжет**: *Научись двигать функции по координатной плоскости как мебель в комнате.*

**Концепции**:
- f(x) + c — сдвиг вверх
- f(x) - c — сдвиг вниз
- f(x - c) — сдвиг вправо (внимание: минус!)

```
│         │    ╱     ╲      ╱   ╲                                │
│    ─────┼───╱───────╲────╱─────╲─────                          │
│         │  оригинал      сдвиг вправо                          │
```

```python
    if D > 0:
```
