## ğŸ› Known Issues & Solutions

### Issue: Plots loading slowly
**Solution**: First plot after launch is slower (initialization). Subsequent ones are fast.

### Issue: 3D brain not rotating smoothly
**Solution**: Use Chrome browser, enable hardware acceleration, close other tabs.

### Issue: Dashboard shows high memory
**Solution**: Normal if >1 GB after many predictions. Restart if >2 GB.

### Issue: Topographic map looks pixelated
**Solution**: This is normal - it's interpolated. More channels = smoother (when OpenBCI arrives).
